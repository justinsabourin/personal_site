Description

A web application for creating web pages where users can create their own projects. Each project has a full editor, the ability to create, edit and delete files, and its own git repository where a user can stage and commit files.

The application began as a school project which I have continually worked on and kept up-to-date.

Technology
Frontend React, Redux, React Router, Material UI, Ace Editor
Backend Node.js, Express, MongoDB, Memcached, Node Git, passport.js
DevOps Docker, Docker Compose, AWS
